ERROR HANDLING




●1. #DIV/0!
  • pembagian dengan 0
  • pembagian dengan sel kosong
=A1/B1 (B1=0)

Cara pikir: tidak bisa membagi dengan “tidak ada”

=IF(B1=0, 0, A1/B1)

atau
=IFERROR(A1/B1, 0)




●2. #VALUE!
  • tipe data tidak sesuai
="Aurel"+10

Analogi: menjumlahkan teks dengan angka

=VALUE(A1)




●3. #REF!
  • referensi sel hilang / dihapus
=A1+B1 (B1 dihapus)

Solusi: perbaiki referensi, jangan hapus data yang dipakai





●4. #NAME?
  • salah ketik function
=SUUM(A1:A5)

Solusi: perbaiki → SUM





●5. #NUM!
  • angka tidak valid
=SQRT(-1)

Analogi: akar negatif tidak valid di Excel





●6. #N/A
  • data tidak ditemukan
=VLOOKUP("Aurel", A1:B5, 2, FALSE)

Jika data tidak ada:

=IFERROR(VLOOKUP(...), "Tidak ditemukan")




●7. #NULL!
  • range tidak benar
=A1:A5 B1:B5
